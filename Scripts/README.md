# Cybersecurity Scripts

This folder contains smaller, standalone scripts that support cybersecurity tasks such as reconnaissance, automation, data parsing, and password evaluation.

These scripts are practical, lightweight tools designed to showcase efficiency, clean coding practices, and technical understanding.

### Included Scripts
- `password_checker.py` – Evaluates password strength based on length, character variety, and entropy.
- `port_scanner.py` – Performs a basic port scan against a target IP.
- `hash_generator.py` – Generates secure SHA-256 and SHA-512 hashes.
- `ip_lookup.py` – Retrieves host information for a given IP address or domain.

Each script includes:
- Purpose  
- Requirements  
- Usage
